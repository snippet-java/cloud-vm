CREATE AND SHARE CONTENT

OFFICE TOOLS ON THE CLOUD
IBM BOX is not only a files sharing system. It allows also create and edit text documents, spreadsheets and presentations. Now you are going to create various documents in IBM BOX.

Objectives
1. Create a New Text Document
2. Create a New Spreadsheet
3. Create a New Presentation


Exercises

1. Create a New Text Document

Here you are going to create your first text document at IBM Box.


STEPS

a) Select a type of document you want to create.


b) Provide file name.


c) Work with your Word document.




2. Create a New Spreadsheet

Now you will create your spreadsheet and exlopre features that system provides.


STEPS

a) Click on NEW in order to select the type of document you want to create. Choose Excel SpreadSheet.


b) Provide file name.




c) You should be able to start editing your data.




d) You can save you work choosing a right option after you click on FILE tab.




3. Create a New Presentation

Now you are going to create your first presentation at IBM Box.


STEPS

a) Click on NEW in order to select the type of document you want to create. Choose PowerPoint Presentation.


b) Provide file name.


c) Now you can start building your presentation.
